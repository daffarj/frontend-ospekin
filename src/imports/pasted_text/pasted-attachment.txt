# OSPEKIN.UB — Stitch Prompts (All Pages)

## GLOBAL DESIGN SYSTEM
> Paste ini di awal setiap sesi Stitch sebagai konteks global

```
Project: OSPEKIN.UB — ospek attribute package ordering platform for new 
students at Universitas Brawijaya, Indonesia.

GLOBAL DESIGN RULES (apply to all pages):
- Framework: React + TypeScript + Tailwind CSS + shadcn/ui
- Install: framer-motion, lucide-react

COLOR SYSTEM:
- Background dark (public pages): #030303
- Background light (dashboard pages): #f8f9fa / white
- Primary accent: indigo (#6366f1) to rose (#f43f5e) gradient
- Text primary dark: white / white-80
- Text primary light: #111827
- Muted text: #6b7280
- Border: rgba(255,255,255,0.08) on dark / #e5e7eb on light
- Success: #10b981, Warning: #f59e0b, Error: #ef4444, Info: #3b82f6

TYPOGRAPHY:
- Font: Inter or Geist Sans
- Headline: font-bold, tracking-tight
- Body: font-light, tracking-wide
- Code/path: font-mono

COMPONENT STYLE RULES:
- Buttons: rounded-full, generous padding
- Cards: rounded-2xl, subtle border, no heavy shadow
- Inputs: rounded-xl, border focus ring in indigo
- Badges: rounded-full, small padding
- Everything feels breathable — generous whitespace
- No heavy drop shadows, no gradients on backgrounds except hero
- Hover states: subtle scale or glow only

SHARED COMPONENTS:
1. Navbar Guest — transparent, logo left, slide-tab nav center, 
   Masuk+Daftar buttons right
2. Navbar User — white bg, logo left, page title center, 
   notif bell + avatar dropdown right
3. Sidebar Admin — dark sidebar (#111827), logo top, 
   vertical nav menu, user info bottom
4. Navbar Kurir — white bg, logo left, "Tugas Pengantaran" title, logout right
```
